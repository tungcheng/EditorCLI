## Installation
### Via Package Manager
1. Open Unity Package Manager (Window ‚Üí Package Manager)
2. Click the `+` button in the top-left corner
3. Select `Add package from git URL...`
4. Enter the following URL:
   ```
   https://github.com/tungcheng/EditorCLI.git
   ```
